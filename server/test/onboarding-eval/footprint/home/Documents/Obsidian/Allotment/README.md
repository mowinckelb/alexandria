# The Allotment

A plot I tend, not a factory floor. (see [[Compost/allotment vs plantation]])

How it's laid out:

- **Daily/** — the running log. most of it is compost with a date on it. some of it isn't.
- **Positions/** — things I've actually worked out and will stand on. slow to add to. some marked `unresolved` on purpose — those aren't unfinished, they're *held*.
- **Compost/** — half-thoughts. left to rot down into something. don't tidy these, tidying kills them.
- **Reading/** — lit notes. reactions in the margin, never summaries. if I wanted the summary I'd have read the back cover.

Rules for myself:
- plainness. say the thing.
- a note isn't done because I'm tired of it.
- don't resolve a tension just to feel clean.
