# Works

*Your personal writing and curation — essays, aphorisms, meditations, quotes, reading material. The publishable corpus that's yours but distinct from raw vault.*

*Different from `vault/` (raw, append-only, everything captured), `ontology/` (in-progress thoughts), and `constitution/` (crystallised beliefs). Works is the curated collection — what you've consciously written or chosen to keep.*

*A scholar's library. Use freely.*

## systems/

Reusable Alexandria machinery the Author invents: prompts, filters, extraction moves, rituals, publishing formats, workflows. Each lives as a single markdown file, `<slug>.md`, with `name` and `description` front-matter — see `factory/templates/system-module.md` for the shape.

If a system is only for you, keep it here. If it could help other Authors, publish it (suggested home: `<github-login>/alexandria-modules`, but any public GitHub path works). Once it has a public path, its Marketplace module ID is `github:<user>/<repo>#<path-without-extension>`. Until then, use `local:<github-login>/<slug>` in `~/alexandria/.call_manifest`. The Marketplace surfaces a module the first time `.call_manifest` reports its ID — *use is the contribution*. See `factory/canon/library.md` for the full lifecycle.
