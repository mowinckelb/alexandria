# Shelf

Droplets for the morning brief. One stanza per droplet, separated by `---`.

Picked deterministically — each ISO week the picker reshuffles, weekday indexes in. 7 unique stanzas per week when you have ≥ 7. Empty file → built-in fallback fires.

Add your own crystallised lines, quotes that have stuck, koans, irreversibility prompts. Edit freely; the brief reads whatever's here on its next fire.
