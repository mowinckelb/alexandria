# Core

*Your operating documents. Loaded by the Engine every session.*

- `agent.md` — how you want AI to work with you. Your principles, communication style, working preferences.
- `machine.md` — the Engine's evolving model of working with you. Rewritten each session.
- `notepad.md` — working memory between sessions. Parked questions, accretion candidates, fragments to revisit.
- `feedback.md` — what works, what doesn't. The Engine adapts based on this.
- `filter.md` — your publishing policy. Controls what kinds of content can be published to the Library.

*Source files have clean names. Derivatives (compressed for inference) have underscore prefix and live in this folder. The Engine regenerates derivatives when sources change meaningfully.*
